<br/>    陈明笑道：“很好，目前想要继续扩充妖军，就要涉及到其他的大妖王了，暂时停止扩军。”
<br/>    
<br/>        黑虎拱手道：“遵命！”
<br/>    
<br/>        陈明看向黑蟒，问道：“你手下的暗部怎么样了？”
<br/>    
<br/>        黑蟒道：“我手下的暗部，都是精锐之中的精锐，经过这一次扩军，我已经将手下的暗部全部换成了妖将，总共一千三百二十三人！”
<br/>    
<br/>        陈明点点头，这个数量，应该也差不多了，暗部执行的任务，并不是大规模作战，而是刺杀，刺探类型的任务，所以对实力的要求更高，算是全精锐。
<br/>    
<br/>        现在实力确实是暴涨，不过灵石的消耗也是暴涨，不说别的，只是一个月的军费，就足足需要数十万不止。陈明最开始接触到这个数字的时候，还觉得也不是很高啊，其实当时是陈明的思维还没有转换过来，就好像大妖的寿元，是三百年，作为长寿种，他们的时间是按几十年，甚至是上百年算的。
<br/>    
<br/>        记名弟子的消耗，比妖军还狠，等到这些记名弟子都能够自己炼制闻道境的丹药了，就可以缓一波了，甚至每个月还有得赚。
<br/>    
<br/>        现在，就是要渡过这个时期，陈明道：“这一次将大家都叫到这里来，是有着另外一件大事。你们，应该都知道沙海吧？”
<br/>    
<br/>        黑虎道：“山主说的可是那个称作妖王墓地的地方？”
<br/>    
<br/>        陈明点头，然后道：“沙海之所以被称作妖王墓地，便是因为沙海之中有着一只风妖老祖，风妖老祖的修为，至少在大君之上，不过我已经和风妖老祖达成了协议，可以随意进出沙海，而且，在沙海之中，我发现了一个上古宗门，现在很是适合我们燕山发展，最为重要的是，极为安全。所以，我准备将燕山搬到沙海之中的上古宗门那儿去，你们怎么看？”
<br/>    
<br/>        黑蟒问道：“不知道山主说的迁移，是说所有燕山境内的修行者还是只是我们燕山本部？”
<br/>    
<br/>        陈明疑惑道：“有什么区别吗？”
<br/>    
<br/>        黑蟒道：“若是全部迁移，需要的时间可能有点长.....”
<br/>    
<br/>        陈明问道：“需要多久？”
<br/>    
<br/>        黑蟒道：“十年.....”
<br/>    
<br/>        陈明惊了又惊，“需要十年？”
<br/>    
<br/>        黑蟒道：“若是全部迁移，那些世俗国家的凡人都要带上的话，十年，可能还不够.......”
<br/>    
<br/>        陈明挥手道：“太久了，只带修行者过去，凡人，谁没事会去杀凡人？而且，燕山只要不倒，他们凭什么敢去杀我们燕山境内的凡人？”
<br/>    
<br/>        黑蟒继续道：“若是全部的修行者，可能会出一些事，比如一些小国家，最高修为的，也只有闻道境的修行者，他们从沙海之中，回到自己原来的国家，可能需要半年多的时间，很难调度，而且，还要他们这半年在路上不出事.......”
<br/>    
<br/>        陈明伸手摸了摸下巴，皱眉道：“也就是说，现在沙海之中的领地，对于我们来说，根本就是只有入道境才能够过去的地方。”
<br/>    
<br/>        黑蟒道：“不错，闻道境毕竟没有法力，不能御剑飞行，赶路还需要看地形，翻阅千山万林，太麻烦了，就好像，现在的南境诸国，还有很多国家根本不知道现在是归燕山统领，他们还打着妖月王庭的旗号........”
<br/>    
<br/>        陈明深吸一口气，这就是发展的速度太快的问题啊，发家致富全是靠抄家这种事，不是慢慢积累起来的。
<br/>    
<br/>        对了，陈明忽然想起来在御剑宗看见过一些奇怪的山峰，看起来似乎没有半点用处，但是山顶上都有着巨大的灯塔，感情好这是指路的啊！
<br/>    
<br/>        就是为了在宗门之中飞行不迷路。
<br/>    
<br/>        陈明道：“所有宗门，势力，入道境以上，除了照看世俗的人，照看门下闻道境弟子的人，其他人，全部迁移到沙海之中，燕山本部，全部迁移！”
<br/>    
<br/>        黑蟒道：“燕山本部迁移倒是不需要多久，燕山的命令执行比其他宗门可是快得多，三个月之内可以完成，但是其他已经归降的妖王什么的，估计需要半年的时间。”
<br/>    
<br/>        陈明现在也只能点头了，毕竟是整个燕山境内的大事，半年，他已经觉得可以接受了，这还是建立在只迁移能够御剑的入道境。
<br/>    
<br/>        燕山的记名弟子倒是不用担心，他们待遇不一样，直接下令让血衣剑卫护送就好。
<br/>    
<br/>        三个月的时间一晃而过，麾下的入道境弟子数量已经突破了三百，让陈明也是有些欣慰。这最早的一批记名弟子，算起来也是一年多的时间了，放在外界，说出去绝对吓死人，放在古仙门，那也是能够小吹一波的。
<br/>    
<br/>        毕竟，陈明可是拿了所有抄家来的财富，全部砸在了闻道境的修行之上，就是古仙门的弟子，也没有这么奢侈。
<br/>    
<br/>        不过也对，陈明相当于拿堪比数个大妖王稳稳当当过几百年的灵石，准备全部在两年之内花完，这种花灵石的速度，实在是有些过于惊人。
<br/>    
<br/>        这一段时间，陈明也没有闲着，现在毕竟只是炼丹大师的造诣，三个月之内，也只是和陈灵鱼写出了一本《华灵丹炼制步骤详解》。
<br/>    
<br/>        毕竟是入道境需要的丹药，不像是闻道境的丹药，随随便便就弄出来了。
<br/>    
<br/>        不过陈明现在并不想让这些新突破到入道境的弟子全部去炼丹，他想让最早突破的六百名记名弟子，全部去炼器。
<br/>    
<br/>        燕山三万血衣剑卫，日后可能是十万，杀王剑和灵甲上面，可是一个大缺口。
<br/>    
<br/>        这三个月的时间，陈明也是终于写出了《杀王剑炼制步骤详解》，其中精确地规定了每一个符印，总共四种符印，锋锐，坚韧，法力贯通，枷锁。
<br/>    
<br/>        锋锐是为了飞剑能够锋锐，坚韧是为了保护飞剑和锁链，法力贯通自然不用说，灵兵都需要这种，枷锁也不难理解，这个符印，就是为了限制妖王的行动力的，法力灌注其中，可以通过这个符印，形成封锁妖王法力流转的速度，虽然不多，但是胜在数量多，量变产生质变。
<br/>    
<br/>        这些记名弟子，也是按照需要绘制的符印数量不同，学习了符印的绘制。
