　　萧逸手握紫云精铁剑，一剑击向顾长风。
<br/>　　只见顾长风手上蓦地出现了一把利剑，利剑清风环绕，甚是不凡。
<br/>　　锵的一声，萧逸被微微击退。
<br/>　　“剑武魂。”萧逸目光一凛。
<br/>　　剑武魂，与刀武魂并列，属于器武魂中攻击力最强的武魂种类。
<br/>　　能觉醒出剑武魂或者刀武魂的武者，一般天赋都很强。
<br/>　　而观这顾长风手中的剑，似乎品阶不低，起码是绿色巅峰武魂。
<br/>　　“我想起来了，你们四只蝼蚁。”顾长风看了眼萧逸以及林劲等人，忽然想起了什么，脸色变得极为难看。
<br/>　　“数天前，我手下胡海，就是被你们所杀，还抢了本应属于本公子的近百点分数。”顾长风说着，脸上布满了杀意。
<br/>　　“是我们又如何？”萧逸冷笑一声。
<br/>　　这几天在森林内，萧逸早就听说过不少关于这顾长风的事。
<br/>　　此人性格狂妄，又锱铢必报，心胸极为狭窄，加之心狠手辣。
<br/>　　所以，今日在看到秦飞扬被围攻时，便立刻出来援助。
<br/>　　一是对秦飞扬很有好感，二则是知道顾长风肯定会找自己四人麻烦，倒不如现在就解决掉他。
<br/>　　“哼。”顾长风冷哼一声，“既然如此，今日本公子便将分数夺回来，顺便送你们四人下黄泉。”
<br/>　　“有本事尽管来。”萧逸怡然不惧。
<br/>　　“饮血回风掌。”顾长风大喝一声，手中一掌打出。
<br/>　　正是刚才打伤林劲和铁牛的那一招血掌。
<br/>　　此乃玄阶巅峰武技，以先天九重的修为打出，威力强横至极。
<br/>　　这就是大家族子弟的好处，拥有别人梦寐以求的高阶武技，战力极强。
<br/>　　不过刚才林劲那杀神一刀也是玄阶巅峰，想来林劲也不是普通人。
<br/>　　言归正传，萧逸没有硬接，而是脚步一动，瞬间消失在原地。
<br/>　　“好快。”顾长风瞳孔一缩，眼睛明显跟不上萧逸的身影，更别说打中萧逸了。
<br/>　　锵，忽然，一声剑吟从顾长风身后出现。
<br/>　　一道剑罡以极快的速度破空而来。
<br/>　　“不好。”顾长风感受到背后的锋芒气息，吃了一惊，连忙转身抵挡，却还是慢了一丝。
<br/>　　只见他手中之剑微微一挡，堪堪让剑罡偏离了自己的要害，却打中了自己的肩膀。
<br/>　　“该死，这家伙的速度怎么会这么快。”顾长风眉头紧皱。
<br/>　　锵，又是一声剑吟。
<br/>　　剑罡再次在他肩膀上画出一道血痕。
<br/>　　十数秒过后，他的肩膀已经鲜血淋漓。而他却连萧逸的衣角都没碰过。
<br/>　　“废物，你只敢躲躲藏藏地偷袭我吗？”顾长风怒不可遏，“有本事出来与我正面交锋。”
<br/>　　嗖嗖嗖....
<br/>　　萧逸的身影不断跳跃着，极快的速度下，让他看起来如同消失了一般。
<br/>　　听着顾长风的话，萧逸心头冷笑一声。
<br/>　　清风剑罡，本身就是飘逸灵动的剑法，不擅力量，但在速度和躲避方面极强。
<br/>　　没理由自己要以己之弱，攻敌之强，除非自己是傻瓜。
<br/>　　再者，他现在不想用紫炎，或者以易霄身份时使用过的手段罢了。否则，就算正面刚，这顾长风也是被秒杀的份儿。
<br/>　　锵、锵、锵....
<br/>　　剑吟之声不断闪过，顾长风不断受伤。
<br/>　　另一便，林劲、铁牛、嫣然、秦飞扬四人轻松地应付着那数百武者。
<br/>　　那数百武者虽然人数多，但都是先天四重和五重罢了，甚至有些只有先天一重、二重，根本不是他们四人的对手。
<br/>　　不过，他们占着人多，打不过了，便立刻后退，由其他人围上来，倒暂时没有出现死亡，顶多重伤罢了。
<br/>　　“咦，那顾长风看起来不靠谱阿。”林劲一刀刀劈出，轻松应付着敌人，闲暇时看一眼萧逸和顾长风的战斗。
<br/>　　“确实。”铁牛也说道，“那顾长风，似乎实战经验非常烂。”
<br/>　　“萧逸兄弟的速度虽然极快，但毕竟只有先天三重的修为，论真正战力，是比不过先天九重的。”
<br/>　　“若换一个经验丰富的先天九重，很轻易就能想到办法破解他的速度优势。就算无法压制他，起码也不至于这般无力。这顾长风，却简直是一面倒地被压制。”
<br/>　　这时，秦飞扬不屑地说道，“你们以为那顾长风算什么东西？”
<br/>　　“他能觉醒出剑武魂，天赋倒是不错。不过他的修为嘛，完全是靠天材地宝和丹药堆上来的。”
<br/>　　“也亏得他有个好家族，大把修炼资源养着他。”
<br/>　　“只是，这样他的根基不稳，在同等级武者的战斗中，显得很弱。只能靠修为优势，对付修为比他低的人了。”
<br/>　　“像我，比他低了一重，便不是他的对手。可是遇到萧逸兄弟这样的硬茬，实战经验丰富，又善速度，他完全没有办法。”
<br/>　　秦飞扬的话音落下不久，果然，另一边的战斗中，顾长风已经因失血过多，渐渐撑不住了。
<br/>　　“该死。”顾长风愤愤地谩骂着，“萧逸是吗？你给我等着。”
<br/>　　说罢，只见他化去了自己凝聚的剑武魂，而后手中漠然出现了另一把剑。
<br/>　　几乎是那剑出现的瞬间，他身上气势暴增，一股锋芒的剑气在他身边数十米范围肆虐，瞬间将萧逸弹开。
<br/>　　萧逸被弹开十数米，站稳身子后，自语道，“竟然是灵器。”
<br/>　　“是顾家的下品灵器，风吟剑。”秦飞扬大喝了一声，“萧逸兄弟，快退，你没有灵器不是他的对手。”
<br/>　　“灵器？”萧逸冷笑一声，而后不退反进，全力一拳轰出。
<br/>　　“哼。”顾长风脸色一冷，道，“现在还不是和你们全力相拼的时候，暂且饶你们几只蝼蚁一命。”
<br/>　　说罢，手中风吟剑一挥，无数道剑气澎湃而出。
<br/>　　在他眼中，已然不将萧逸当成威胁，而是打向林劲等人。
<br/>　　萧逸一惊，他自然不惧这些剑气，却怕林劲等人挡不住，只能立刻后退，回到林劲等人身边。
<br/>　　“退。”同一时间，顾长风大喝了一声。
<br/>　　那数百武者立刻撤离。
<br/>　　萧逸刚想留下他们，无数剑气已然打来，只得挡下剑气，眼睁睁地看着他们离开。
<br/>　　当剑气悉数被挡下后，数百武者早已远远逃离，还有数十个修为较弱的逃得慢，比萧逸和林劲拦了下来。
<br/>　　“饶命阿。”那数十武者顿时跪地求饶。
<br/>　　“饶命？”林劲冷声道，“刚才你们可是打算置我们于死地。”
<br/>　　“死吧。”林劲冷哼一声，一刀劈出。
<br/>　　“我们认输。”数十武者，顿时惊恐地叫了一声，而后手中令牌一扬，一道虚无的光罩笼罩了他们的身体。
<br/>　　嘭嘭嘭，林劲的刀，轻易被挡下，数十武者则完好无损。
<br/>　　这是令牌上的禁制，也是裂天剑派给予的保护。
<br/>　　按照规则，参加考核者，若遇生命危险，可触发这层禁制，保自己周全。
<br/>　　但触发禁制，便代表着认输，考核失败，半个时辰后，会有剑派内的执事将其带离森林。
<br/>　　禁制的强度，起码要洞玄六重以上的武者才能打破。
<br/>　　只是，不到迫不得已，没人愿意使用。
<br/>　　因为一旦用了，就无缘成为裂天剑派的弟子了。
<br/>　　而现在这十数个跪地求饶的武者，明知林劲要杀他们，只能被迫使用了。
<br/>　　当然，也有的武者没来得及用就死亡。
<br/>　　像之前的胡海等人，本身就和萧逸他们差距很大，又仗着自己人数多，觉得可以击杀萧逸他们，没来得及使用禁制就被秒杀了。
<br/>　　其实，这是一个考核，而并非生死仇杀。一般参加考核的武者，不会随便杀人，取了分数后，便将人放了。
<br/>　　但像胡海，还有顾长风等人，心狠手辣，取了分数还要杀人，萧逸等人自然不会留手，自然也要取他们性命。
<br/>　　言归正传。
<br/>　　这十数人使用禁制后，令牌上的分数会自动出现在光罩之上。
<br/>　　萧逸只需将自己的令牌放到光罩边缘，会自动获得他们的分数。
