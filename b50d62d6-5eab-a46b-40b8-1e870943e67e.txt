　　数天后，又是议事厅中。
<br/>　　“五长老，紫云铁已经为你打理好，直接运回家族即可。当然，你若想继续待在矿脉里看看陨星山脉的风景，我们也欢迎。”
<br/>　　四长老客气地说道。
<br/>　　虽然前几天五长老大闹了一场，但毕竟同为萧家之人，四六长老和执事们并没有往心里去。
<br/>　　五长老阴沉着脸，想必是余怒未消，不冷不淡地说道，“如今天色尚早，不急。矿脉中的紫叶灵茶别有一番味道，老夫多品尝几杯再离去不迟。”
<br/>　　四长老笑道，“五长老喜欢，便多喝些。待会儿我命人为你多准备些，拿回家族品尝也是好的。”
<br/>　　“有劳了。”五长老随意拱了拱手。
<br/>　　“客气。”四长老笑着摆摆手。
<br/>　　气氛忽然静默了一下。
<br/>　　五长老继而叹了口气，“老四、老六。”
<br/>　　“额。”四长老和六长老同时愣了愣，惊讶于五长老没有称他们为长老。
<br/>　　其实，很多年前，萧家内部还没有纷争之时，九大长老间的关系还是非常亲密的。彼此也不会长老长老这般生分地称呼，而是老三、老四、老五、老六这样打招呼。
<br/>　　那时候的萧家，家主萧晨枫尚未失踪，家族势力处于最巅峰之时，比另外两大家族更为强盛，可谓紫云城之霸主。
<br/>　　只是，后来随着一系列事情的发生，家主失踪，大长老和二长老相继少理族中之事，其余长老也互相因权力之争而产生了间隙。
<br/>　　五长老蓦然一笑，道，“记得很多年前，我们正值壮年时，也如现在的家族子弟般，前来陨星山脉历练。”
<br/>　　回忆起当年的峥嵘岁月，四六长老也是会心一笑。
<br/>　　“对阿，当年的陨星山脉，可比现在乱得多。其他势力有不少都想打我们萧家矿脉的主意呢。”
<br/>　　“我们为了家族的尊严和荣耀，以及利益，誓死与他们战斗。还记得慕容家和江家那些混蛋，一个个都被我们收拾得很惨呢。”
<br/>　　“哈哈，那时家主也尚年轻，带着我们横扫陨星山脉，无人能敌。现在慕容家和江家那些长老们，哪一个没有被我们收拾过，想起来真是开心。”
<br/>　　四六长老谈起往事，很是开心，和五长老聊得特为有劲。
<br/>　　也不知为何，今日的五长老竟有这番兴趣料及往事，一杯杯茶水入腹，丝毫没有要离开的意思。
<br/>　　“老四、老六，你们就记得家主带领过我们。难道就忘了，我们曾经也一起并肩战斗，甚至生死相托吗？”五长老忽然叹了口气。
<br/>　　四长老道，“老五，那些我们当然记得。我还记得，有一次我们几人到了陨星山脉深处，遇到了一群妖兽袭击，要不是你机灵，我和老六差点都交代在了那里。”
<br/>　　“既然记得。”五长老忽然脸色一变，语气也变得愤怒，“你们为何帮着萧逸那个小废物，处处与我作对，竟丝毫不念当年的情分。”
<br/>　　“额。”四六长老忽然一惊，没想到五长老忽然发怒。
<br/>　　一旁的萧逸原本在静静地听着三位长老聊天，此时也皱了皱眉头。
<br/>　　“老五，你先别生气....”四长老说着。
<br/>　　“住口。”五长老一声大喝，“你们不配这样叫我，既然你们忘了当年的情分，丝毫不当我是萧家族人，我也不必与你们客气，更不必对你们心软。”
<br/>　　“老五，我们何曾不当你是萧家之人...”四六长老刚想解释一番，忽然脸色大变。
<br/>　　“五长老，你....噗。”四、六长老同时吐了一口鲜血。
<br/>　　各大执事们，包括萧逸，也是脸色难看，而后口吐鲜血。
<br/>　　“这茶水有问题。”萧逸脸色一边，看了眼自己的杯子。
<br/>　　其余人也纷纷想到了这一层，看向了杯中的茶水。
<br/>　　四长老脸色发黑，惊疑地看向五长老，“怎么可能，紫叶灵茶一向有专人看管，你没机会下手。”
<br/>　　“哈哈哈哈。”五长老大笑道，“茶自然是没问题的，但这水嘛....呵呵。”
<br/>　　四长老顿时大怒，“你竟然在水里下毒？那岂不是整个矿脉不单止我们，所有族人都中毒了？”
<br/>　　六长老盛怒下直接准备出手，然而，他刚准备调动体内真气，却猛地身子一软，瘫坐在原地。
<br/>　　“怎么可能，我的真气竟然悉数被压制了，半分功力都使不出来。”六长老惊怒交加。
<br/>　　“哈哈哈哈，没用的。”五长老大笑道，“在两个时辰内，你们都休想发挥出修为，跟个废物没什么区别。”
<br/>　　“不可能。”四长老惊疑道，“我和老六乃是先天武者，真脉养护全身，不可能有毒药能压制我们的修为。”
<br/>　　一旁的萧逸咬牙道，“不是毒药，是武魂的力量。”
<br/>　　他万万没想到，精明谨慎如他，竟也着了五长老的道，此刻真气完全被压制，全身无力。
<br/>　　他发现，自己体内有一股莫名的黑气，黑气自主运转，而后凝结成一朵漆黑如墨的花骨。就是这朵花骨，压制了自己的真气。
<br/>　　很显然，这不是毒药，而是武魂的力量渗透进了体内。
<br/>　　“五长老，你好卑鄙。”萧逸冷冷地看着五长老，目露凶光。
<br/>　　他怎么都想不到，五长老竟丧心病狂到这个程度，连同族的其他人都不放过，歹毒如斯。
<br/>　　当然，其实这也不怪他。毕竟，矿脉里有两位先天长老。
<br/>　　先天武者的强大，给了他足够的自信。
<br/>　　先天武者的身体已经强悍到恐怖的地步，绝对没有任何外物能伤到他们，哪怕是三品丹药中的毒丹都不行。
<br/>　　能伤到先天武者的，只能是比先天武者更强悍的实力。
<br/>　　也是这个原因，萧逸才不惧五长老会耍计谋，更没有提防。
<br/>　　四六长老和众执事们也是如此。
<br/>　　这时，四长老听到萧逸的话，顿时脸色大变，“武魂的力量？紫云城中，武魂具有毒性，且能压制他人修为的，只有一个。”
<br/>　　“慕容墨，慕容家多年未曾露面的大长老。”六长老咬了咬牙。
<br/>　　“慕容墨？”萧逸也是一惊，他听说过这个名字，多年前也是名震紫云城的大人物.
<br/>　　不过此人已经很多年未曾露面了，有人说他是退居幕后了，也有人说他是离开了紫云城，众说纷纭，无人真正知晓。
<br/>　　“五长老，你竟敢勾结慕容家的人？”四长老面露杀意。
<br/>　　以往，五长老就算以权谋私，打压萧家族人，其他长老都念着他对家族有功，睁一只眼闭一只眼。
<br/>　　但，勾结外人，残害同族，此乃萧家族规的禁忌。
<br/>　　长老和执事们此刻已经起了杀意，但奈何真气被压制，只能目光冰冷地剐着五长老。
<br/>　　面对众人冰冷的目光，五长老丝毫没有在意。
<br/>　　“不要怪我。”五长老冷声道，“要怪就怪你们自己，你们不仁，也休怪我无义。”
<br/>　　“哈哈哈哈，说得好。”
<br/>　　议事厅外，忽然一道阴鸷的笑声传来，两道人影来到了议事厅中。
<br/>　　“慕容槐，慕容蝎？”四长老咬牙切齿地说着。
<br/>　　来人竟然是慕容家矿脉的两个长老。
<br/>　　“五长老，成大事者，就该有这个气魄。”慕容槐阴声笑道。
<br/>　　“今日，萧家的人，一个都别想活命。”慕容蝎得意地说道，阴冷的目光在所有萧家之人身上扫过。
<br/>　　在扫视到萧逸身上时，目光尤为恶毒。
<br/>　　“该死，这两个慕容家长老竟也来了。”萧逸心头顿时大急。
