　　毫无疑问，两道噬火百刃就算同时打出，仍旧没有任何卵用。
<br/>　　银背妖蟒的防御力，连一般的先天武者都打不破，更被说萧逸这个区区后天境了。
<br/>　　“再这样下去，我的真气耗光，这条孽畜都不会受伤，我必死无疑。”萧逸虽然着急，但还是镇定地思索着对策。
<br/>　　他现在只是靠着豹形和蛇形，以及武魂力量的结合，还有真气，这才发挥出远超自己修为的速度，勉强保持不败。
<br/>　　他甚至已经放弃了噬火百刃的攻击，节省真气。
<br/>　　“若无法打破这孽畜的皮肉，我没有胜算。”萧逸心中想着，同时身子一动，再次躲过了银背妖蟒的攻击。
<br/>　　银背妖蟒早就在暴走状态，它的疯狂攻击，已经将周围百米范围毁于一旦。
<br/>　　它现在只有一个想法，迫切地想要将面前这个灵活的人类杀死。
<br/>　　“破开它的皮肉...破开它的皮肉....”
<br/>　　萧逸心头不断想着。
<br/>　　他发现，就算自己心智过人，沉稳无比。可在实力上远远比不上银背妖蟒，始终无计可施。
<br/>　　他头一次发现，在这个世界中，武力到底重要到什么地步。
<br/>　　“若这次能逃脱，一定要将武道修炼放到第一位，实力，才是这个世界生存的资本。”萧逸暗暗想着。
<br/>　　即便在这种危机状态，他仍旧没有慌。
<br/>　　正当他的真气几乎耗尽，体内控火兽武魂也因过度使用而承受不住时....
<br/>　　“不对，我还有冰鸾剑武魂，它或许可以破开这孽畜的皮肉。”萧逸忽然眼前一亮，想到了应对的办法。
<br/>　　冰鸾剑武魂，一直是他保命的底牌，更是他不能随意展露出来的东西。他几乎没有用于战斗，渐渐忘了它的存在。
<br/>　　这危急时刻，他却立刻想了起来。
<br/>　　嗖，萧逸身子一动，消失在原地，而后出现在银背妖蟒之上。
<br/>　　一道深紫色的亮光凭空出现，那耀眼的光芒，仿佛天地日月也将因它而黯然失色。
<br/>　　一把古朴却又散发着骇人寒芒的长剑，出现在萧逸手中。
<br/>　　几乎是冰鸾剑出现的瞬间，本来处于暴走状态的银背妖蟒立刻打了个冷颤。
<br/>　　那恐怖的寒芒，似乎能将它的妖血全部冻住；那骇人的锋芒，仿佛能将它送入黄泉。
<br/>　　萧逸一剑刺下。
<br/>　　撕拉一声，如同刺破薄纸般简单，银背妖蟒的皮肉瞬间被刺穿。
<br/>　　萧逸手握冰鸾剑，身体一跃，往地下跳去。
<br/>　　冰鸾剑在他的带动下，沿着银背妖蟒的背部，一直划过。
<br/>　　当他落到地面时，这头三级妖兽银背妖蟒，已经被分成两半。
<br/>　　萧逸再次一动，将那蛇头也一并砍下。
<br/>　　数十秒后，原本强大无比的银背妖蟒，抖动了多下之后，彻底死亡，成为陨星山脉中又一具冰冷的尸体。
<br/>　　“冰鸾剑，你果然没有让我失望。”萧逸呼了口气。
<br/>　　萧逸随手一挥，从妖蟒尸体上取出了一颗鹅卵蛋大小的珠子，这就是妖兽的内丹，妖兽最值钱的部分。
<br/>　　萧逸还想着日后用它来炼制金脉丹。
<br/>　　三级妖兽内丹到手，萧逸没有再管尸体，而是散去冰鸾剑，而后立刻跑到晕厥过去的萧景等人身旁。
<br/>　　大致检查了众人的伤势后，萧逸的脸色顿时变得阴沉如水。
<br/>　　所有人都是重伤。
<br/>　　萧景是后天七重武者，肉体更为强悍，喂食了丹药后，不多时便清醒了过来。
<br/>　　之前为家族子弟炼制疗伤丹药时，萧逸特地留了一手，炼制了一些二级疗伤药，效果极佳。
<br/>　　“咳咳。”萧景咳嗽了两声，吐了口腥血，渐渐稳下了伤势。
<br/>　　当他看到面前的景象后，顿时倒吸一口冷气。
<br/>　　方圆百米如同末日废墟一般，树木被全毁，地面破碎，特别是看到银背妖蟒的尸体时，更是惊讶至极。
<br/>　　“少...少家主，这...这是怎么回事？”萧景问道。
<br/>　　萧逸没有回答他的话，在陆续为其他人喂服丹药后，他坐到了萧壮和萧子木两人身旁。
<br/>　　其他人都陆陆续续醒了，只有这两人，始终昏迷。
<br/>　　“慕容千军，铁刀猎妖队。”萧逸咬着牙低语了几句，满脸杀意。
<br/>　　“少家主，萧壮和萧子木怎么了？”萧景见萧逸脸色不对，立刻问道。
<br/>　　萧逸咬牙道，“他们的伤势稳住了，暂时不会有生命之危，但他们的丹田被毁了。”
<br/>　　“什么！丹田被毁了？”萧景脸色一变。
<br/>　　丹田，可是武者最重要的东西。一旦被毁，代表着修为全失，甚至日后再无修炼的可能，成为一个废人。
<br/>　　之前这两人死死抱住蛇尾，最是视死如归，也伤得最严重。
<br/>　　“萧景执事，你现在的伤势怎么样？”萧逸沉声问道。
<br/>　　萧景回答道，“基本没有大碍了，七条主脉同时吸收天地灵气，并治愈自身，很快就能恢复实力。”
<br/>　　“那就好。”萧逸阴沉如水，道，“待会儿，要麻烦你护送他们二人回去了。”
<br/>　　“额，好的。”萧景下意识地回答一句，而后一惊，问道，“少家主你呢？”
<br/>　　萧逸从乾坤袋中拿出一把丹药，大口地扔入口中，快速地恢复体内真气。
<br/>　　“慕容千军，铁刀猎妖队，一个都别想活着走出陨星山脉。”萧逸冷冷地说了一句，脚步一动，消失在原地。
<br/>　　......
<br/>　　数十里之外，慕容千军和铁刀得意地走着。
<br/>　　“铁刀队长，这次得你帮助，我才有机会得到这三颗妖血炎心果，回到矿脉后，我必有重酬。”慕容千军笑道。
<br/>　　铁刀虽然是猎妖队队长，更是后天六重武者，但在慕容千军面前，还是得毕恭毕敬的。
<br/>　　“千军公子客气了，你乃不世之才，就算没有我的帮助，我相信你取那三颗妖血炎心果也是轻而易举的。”铁刀客气地说道。
<br/>　　“哈哈哈哈。”慕容千军狂傲一笑，对于铁刀阿谀奉承很是受用。
<br/>　　“不管怎么样，这次你有大功。”慕容千军道，“此番不单止得了妖血炎心果，还顺带将萧家那群笨蛋给了结了。特别是那萧逸....”
<br/>　　慕容千军道，“原本我以为，这次能干掉萧景这个精英执事已经很不错了，没想到连萧逸也一并对付了。我听说他可是萧家最近才崛起的天才，连萧若狂两兄弟都败于他手上。”
<br/>　　铁刀道，“萧逸现在肯定已经葬身蛇腹了，那等废物，岂能与千军公子相提并论。”
<br/>　　“就是，那等废物，给千军公子提携都不配。”铁刀猎妖队的队员纷纷奉承着。
<br/>　　“哈哈哈哈。”慕容千军大笑着，“如果有机会，真想与他交手一番，可惜他已经死了。”
<br/>　　恰在此时，远处一道破空声袭来，一道暴喝一声响彻所有人的耳朵。
<br/>　　“慕容千军，我给你这个机会。”
<br/>　　“嗯？”慕容千军一惊，寻声望去，只见一道火红色身影疾速袭来。
<br/>　　“是萧逸。”慕容千军皱起了眉头。
<br/>　　铁刀沉声道，“这小子好高超的密林追击手段，我们走了那么远，密林内又弯弯曲曲的，他竟然能追上我们？”
<br/>　　“不对！”铁刀反应过来，顿时一惊，“他是怎么逃出来的，没人能在银背妖蟒的追击中活命。”
<br/>　　“将死之人，何必多问。”萧逸转瞬间来到，两道火刃同时打向慕容千军和铁刀。
<br/>　　嘭、嘭，火刃爆炸，瞬间将两人震退。
<br/>　　“好强的攻击，这小子不是才后天一重的修为吗？”铁刀吃了一惊。
<br/>　　萧逸挺身而进，没有管这两人，直接朝其他猎妖师攻去。
<br/>　　这些猎妖师，充其量不过凡境九重，哪里是萧逸的对手。
<br/>　　萧逸一记虎形，轻松将一个猎妖师划破了喉咙。
<br/>　　在火焰之下，大片热血从喉咙中喷射而出，喷湿了萧逸的脸庞。
<br/>　　但，萧逸没有在意。
<br/>　　豹形一动，身体快速在这数十个猎妖师中游动。
<br/>　　不过数秒，十数人全部被划破了喉咙，成为了一具尸体。
<br/>　　萧逸的双手，沾满了他们的鲜血。然而，这些鲜血，并没能浇灭他心头的怒火；浓郁的血腥味，更不能掩盖他惊天的杀意。
<br/>　　慕容千军和铁刀顿时一惊，他们万万想不到，萧逸竟能在这么短时间内用最快的速度杀光了十多个猎妖师。
<br/>　　“咕。”两人同时咽了口口水，下意识地退后两步，在他们眼中，浑身是血的萧逸，如同地狱中走出来的杀神，让人惊惧。
