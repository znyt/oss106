　　第193章
<br/>　　雨下起来没完没了，去年青牛山没有降雨，老天仿佛是将去年一年的雨量加到今年，大雨顺着山岭哗哗冲下来，干涸的小溪河流都被添满。
<br/>　　大雨阻隔着无法进攻，戎族倒是冒雨尝试过一次进攻，结果弓不能发，马不能行，投石机达不到射程，士兵被雨幕打着睁不开眼睛，好不容易挤到城门外，迎来的是蛤蟆寨的滚木檑石还有火药，最后伤亡惨重。
<br/>　　营地内还要防涝，将积雨排出去，经常有帐篷被雨水压倒，因为天气太过潮湿的缘故，还有不少人得了褥疮。
<br/>　　大家都憋着一股劲，可这雨时停时不停的，大王子每天看着天空，一见太阳出来，忙排兵点将，誓要一举打下蛤蟆寨，结果大兵刚点好，雨就哗啦啦的下来。
<br/>　　众人一腔战意，被大雨淋得湿透。
<br/>　　他们是陷在了青牛山，那种大概叫做『你是个废柴』的气场包裹了他们，智商削弱，武力削弱，骑兵天赋被限制，运气减半。在外面，他们攻城略地，如入无人之地，可一旦进入青牛山，他们身上就贴了这种Buff。
<br/>　　这一日，阴天，小雨。
<br/>　　巫医萨特曼来到大帐中，刚进入大帐他就吓了一跳，只见北蛮部的九位王子，用一种『他好帅我好想和他发生点什么』的眼神看着自己。
<br/>　　萨特曼紧了紧身上的袍子，难道这几位王子里还有喜好男风的。
<br/>　　“长老，据说你可以请神？”大王子问。
<br/>　　萨特曼定了定神，大概判断出对方中某人可能生病了，这可是自己熟悉的领域。
<br/>　　“我信奉永恒的长生天，草原上的神灵都接受我的侍奉，不知是哪位尊贵的王子病了，我可以向天神祈祷，驱走残暴的病魔。”
<br/>　　“我们都没病。”
<br/>　　“……”萨特曼一愣：没病找我来干嘛。
<br/>　　“可会呼风唤雨？”
<br/>　　“呼风唤雨……大概会吧。”萨特曼顿了顿，现在可不好意思说自己不会。
<br/>　　“好！”大王子重重一拍帅案：“你可有把握破掉那贼人的法术？”
<br/>　　比这场大雨更讨厌的，是程大雷每天都会站在城头，披着蓑衣哼哼唧唧。
<br/>　　“苍茫的天涯是我的爱……”
<br/>　　不知为何，每当大王子重复这句咒语时，便会不自觉地哼唱出来。果然是魔咒呐，不知不觉我已受了影响。
<br/>　　“破他的法术……”萨特曼眼睛睁得好大，心里咆哮道：大王子，你清醒些好不好，封建迷信害死人呐。
<br/>　　“哈，我果然没有猜错，想那南人即便有些妖术，又怎能和长生天的信徒相提并论。明日长老就设坛施法，破了他的妖术，哈哈。”
<br/>　　『呃……大王子的意思是我已经答应了，可我什么时候答应过，这中间究竟有什么误会』
<br/>　　次日，城门五百步外立下一高台。萨特曼爬了上去，举着棍子看着城门。
<br/>　　城门上，程大雷披头散发，踩着奇怪的步伐，嘴里大声吟诵着咒语。
<br/>　　“别看我只是一只羊，绿草因为我变得更香。”
<br/>　　萨特曼欲哭无泪，咒语自己都听不懂，果然是个高手。只希望动手时，对方不要打得太狠。
<br/>　　“呔！”萨特曼大着胆子喝了一声，高声道：“我是永恒长生天的虔诚信徒，草原精灵的代言人，丑陋的妖人，不要在我面前卖弄你的愚蠢。”
<br/>　　一声高喊打断了程大雷的节奏，憋得程大雷分外难受，他心里也是一头雾水，心道：这货想干嘛。
<br/>　　不过对方出招了，程大雷也不能不接着，他冲对面大声喊道：“吾乃是南极仙翁亲自传艺的蛤蟆仙人，继承煌煌仙道，尔等化外蛮夷，竟也敢在我面前枉论天道。”
<br/>　　“呔，今日就看我破你的妖术。”萨特曼大喝一声。
<br/>　　程大雷心里有些打鼓，这世界的力量层次自己到现在都没有搞清楚，系统既然能奖励自己风雨，万一这世界真有能呼风唤雨，撒豆成兵的道人呢，水浒里不还有个入云龙公孙胜么。
<br/>　　他睁大眼睛瞧，只见萨特曼举着一杆木棍，身披宽大的羊皮袍子，身体不停抖动，口中念念有词。
<br/>　　“不好！”
<br/>　　程大雷心中一惊，此人步伐玄妙莫测，动作隐隐合着某种节奏，这莫非，莫非是……传说中的东北秧歌。
<br/>　　程大雷舌头有些打结，东北秧歌乃是天下广场舞之祖，自己再以神曲迎敌，怕要被对方克制。若想要胜过对方，除非祭出那招绝世武功——社会摇。
<br/>　　所谓：从来不闻社会摇，纹满小猪也枉然。
<br/>　　“我曾经是个王者，后来说声算了，拱手让位兵马大权还被别人攥着”
<br/>　　萨特曼如陷入魔咒中，身体情不自禁颤抖，一种异样的情绪在胸腔内酝酿，令他魂不守舍，神魂颠倒。
<br/>　　“他曾经年少轻狂，为着江山疯狂”
<br/>　　呀呀呀，萨特曼勉力支撑，可那魔音穿耳入心，令他无法自持。他头晕脑胀，只感觉周围景物已看不清楚，忽然大吼一声，口吐鲜血，跌下高台。
<br/>　　“快救长老，快救长老。”
<br/>　　大王子高呼，此人的法术太可怕了，一旦陷入其中，便头晕脑胀，智商下线。
<br/>　　“明明觉得很难受，偏偏还想听下去。”
<br/>　　“是呐，是呐，还有那动作，我仿佛看到了死神的舞蹈。”
<br/>　　众人看向城头的程大雷，眼睛流露着畏惧：太可怕了，太可怕了。
<br/>　　程大雷拍拍手掌，不信我社会摇和喊麦两大杀器祭出，你还能撑得下来。
<br/>　　哼，打完收工。
<br/>　　大雨，断断续续下了大半个雨，众人在雨声和程大雷的魔音中饱受挣扎。可再漫长的雨季总也有过去的时候，这天中午，连绵的雨水终于停了。
<br/>　　乌云散去，日头冒出来，阳光普照大地，长久的阴霾一扫而空。
<br/>　　北蛮部十万兵丁，人人心里都窝着一口气，终日磨刀霍霍，都盼着这个晴天呢。
<br/>　　当这一天来临时，大王子集结兵力，在蛤蟆寨前排开，九位王子每人一匹高头大马，立在队伍的最前方。
<br/>　　北蛮部九位王子，从未像今日一样团结，而他们之所以能凝聚在一起，是因为一个共同的敌人。
<br/>　　今日不计代价，不计伤亡，一定要打下蛤蟆寨。
<br/>
