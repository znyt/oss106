　　第156章
<br/>　　薛四，就是薛半川的小名。
<br/>　　杨龙停连这都知道，看来他为筹备这一战，早已准备良久，对薛半川进行过认真的调查。相比之下，薛半川连杨龙停与韩虎距的关系都不知道，就显得有些不够格了。
<br/>　　而程大雷对于杨龙停也做过调查，毕竟知己知彼才能百战百胜，来自信息大爆炸时代的程大雷，比任何人都明白情报的重要性。
<br/>　　杨龙停年少从军，是从血海里拼杀出来的，一开始的表现只是普通，说不上差，但也绝对称不上优秀。真正让他扬名的一战，还是在戎族入侵的时候。那时他与韩虎距联手，临绝境，统残兵与戎族血战，随之扬名天下。
<br/>　　但仅靠此，杨龙停还成不了今日的幽州王。杨龙停的发迹还要靠他的夫人，他夫人崔氏乃是京州的大户，在得到崔家的提携后，杨龙停被封为幽州王，韩虎距则只是黑石城城主。
<br/>　　这件事好像在两人之间种下芥蒂，之后三十年，二人从未再有过来往。
<br/>　　你活着我不搭理你，你死了我替你报仇。
<br/>　　杨龙停和韩虎距之间的关系，也颇耐人寻味。
<br/>　　这就是程大雷得到的消息，在幽州城几乎算是半公开的，更深的内幕程大雷也就打听不到了。
<br/>　　仅靠目前的信息，程大雷也能推断出，这杨龙停绝不可轻视。
<br/>　　杨龙停亲自挂帅，统兵五万直扑黑石城，终在这一日，行到青牛山。
<br/>　　“欢迎，欢迎！热烈欢迎！”
<br/>　　徐神机领着一队人站在道路两侧，挥舞着小旗帜，奇怪的是他们口鼻都蒙着白布。
<br/>　　杨龙停都座撵停下，看着眼前这一幕，居高临下道：“在搞什么，程大雷内，为何不出来见我。这次攻打黑石城，我还打算让青牛山陷阵营打头阵，建功立业，日后好提拔他，为何他不出来见我？”
<br/>　　“这是祝愿王上旗开得胜，马到成功。”徐神机将小旗收起来，道：“大当家病了。”
<br/>　　“病了？病得这么巧，该不会是装病吧？”
<br/>　　“不瞒王上，不仅大当家病了，山寨还有很多人得病，现在山寨正闹瘟疫。”
<br/>　　“瘟疫！”
<br/>　　杨龙停被这两个字吓了一跳，行军打仗最怕就是瘟疫，他们要比戎族还恐怖。这个时代还没有抵御瘟疫的有效手段，莫说药物防止，连隔离的手段都不懂。唯一能用得上的法子，大概就是……烧香拜佛了。
<br/>　　真若染上瘟疫，自己这五万大军一朝便会崩溃。
<br/>　　“什么瘟疫，我怎么没听说过此地有瘟疫。”
<br/>　　徐神机压低声音，口吐两个字：“沙司。”
<br/>　　“沙司？”
<br/>　　“意思是，一旦染上了什么人都能杀死。”
<br/>　　杨龙停皱起眉头，他不信徐神机都话，怎么那么巧，自己要用他时，他偏偏就病了。可瘟疫却又不得不防，杨龙停不敢冒险。
<br/>　　“让程大雷来见我，立刻马上。”
<br/>　　“可大当家染了瘟疫，怕……”
<br/>　　“他不来见我，我就去见他。”杨龙停微微挑起眉毛，这个六十岁的老人，毕竟是一州之主，言语间杀意弥漫。
<br/>　　“王上，王上……程大雷来见您了。”
<br/>　　就在这个时候，一个虚弱的声音响起，杨龙停看到程大雷由两个人搀着，有气无力的来到自己面前。
<br/>　　“你真的病了？”杨龙停竟有几分动摇，关键是程大雷现在的样子太吓人了。
<br/>　　脸白得没有半点血色，不时咳嗽，若没人搀扶，站在那里随时都会跌倒。
<br/>　　“末将无病，末将身体好得很……咳咳……这就准备盔甲，随王上出征，报答王上的知遇之恩。”
<br/>　　“有病就当治么，治好了病再保家卫国日后也不是没有机会。军医……唤军医过来，给程将军看病。”
<br/>　　一个背着医囊的郎中过来，望闻问切，与程大雷把脉，查看他的舌苔眼球……
<br/>　　“怎样，他病得严重么？”杨龙停高高在上问。
<br/>　　“禀王上，程将军体寒入虚，怕已伤了肺脉，若不及时医治，恐有生命危险。”
<br/>　　杨龙停暗付：难道他真染了瘟疫，竟是我错怪了他不成。
<br/>　　“再唤个军医过来，你这庸医怎么治得好我程将军的病。”
<br/>　　连换了三个郎中，判断却是一摸一样，程大雷染了极严重的风寒病。
<br/>　　“我没病！我没病！我要上阵杀敌，报效王上……呃咳咳咳！”程大雷推开身边的人，慷慨激昂的喊道。
<br/>　　“好啦，好啦。”杨龙停摆摆手：“有病就养病，日后有你立功的机会。”
<br/>　　废话，得了瘟疫还想往我的队伍钻，想要害死我么。
<br/>　　杨龙停一刻没有停留，继续领兵前进，而程大雷站在原地，泪眼模糊，久久不肯离去。
<br/>　　“大当家，真有你的啊。”徐神机冲程大雷竖起大拇指：“刚才真是吓死我了，你是怎么装病的，三个大夫都没看出来。”
<br/>　　“什么装病！”程大雷重重打个喷嚏：“老子在河里泡了三天，才成功染了风寒，我不是装病，我是本来就有病，我容易么我。”
<br/>　　徐神机吓了一跳：“好，你有病你了不起，来我扶你回去。”
<br/>　　“等下把黄三元叫来，我有事吩咐他去做。”
<br/>　　“什么事啊，还得黄总管专门去做。”
<br/>　　“让他去黑石城，见薛半川，就说……我们要和他合作。”
<br/>　　青牛山介于黑石城和幽州城之间，任何一家消灭另一方后，他们首先做的就是消灭蛤蟆寨。所以，为了山寨的存活考虑，程大雷绝不能让薛半川死。
<br/>　　黄三元由赵子龙护送，避过大军，从小路超过杨龙停他们。在颇费一番周折后，才成功潜入黑石城。首先找到陆哼，再通过陆亨去见薛半川。
<br/>　　再费了九牛二虎之力后，黄三元才算是见到薛半川。
<br/>　　黄三元和薛半川以前就认识，算是曾经的故人。
<br/>　　见到黄三元时，薛半川正在写字，黄三元暗暗佩服，大军压境，竟还能如此镇定，非一般人能做到。
<br/>　　“薛城主，我此次冒死前来，是奉大当家的命令，与薛城主商量一件性命攸关的大事……”
<br/>　　听明白黄三元的来意后，薛半川愣了一会儿。
<br/>　　“合作？我马上要投降了。”薛半川指着摊开的纸笔：“这不……正写降书呢。”
<br/>　　
<br/>(本章完)
