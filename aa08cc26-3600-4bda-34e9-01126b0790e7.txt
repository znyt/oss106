　　呼…
<br/>　　周遭寒风呼啸。
<br/>　　剧烈的冰暴罡风，整整覆盖了百里范围，且吹袭不停。
<br/>　　“嘶。”萧逸微微呼了口凉气，紧了紧自己的宽大衣袍。
<br/>　　寒风呼啸中，他竟觉几分寒冷。
<br/>　　嘭…
<br/>　　萧逸手中一道毒力凝聚。
<br/>　　呼…
<br/>　　寒冷罡风，吹袭而过，吹得那团毒力摇晃数分。
<br/>　　待得罡风吹过。
<br/>　　萧逸手中毒力，并无变化，仍旧那般黑得浓郁，散发着惊人剧毒气息。
<br/>　　但，萧逸却很清楚，这团毒力，已经没用了。
<br/>　　这团剧毒，外表看似无异状。
<br/>　　实则，里头的剧毒，早已化作冰粒，随后自动湮灭。
<br/>　　那吹袭而过的罡风，如若一只冰冷的噬魂鬼魅。
<br/>　　前方，惊蛰剑的身影，已然消失。
<br/>　　不，不是消失了，而是早已从高空跌落在地。
<br/>　　萧逸身影一闪，从高空落下。
<br/>　　惊蛰剑，已是一具尸体。
<br/>　　尸体无损，面部表情，一如死前那般冷漠。
<br/>　　但在萧逸的感知下，惊蛰剑体内，五脏六腑、四肢百骸，包括那一身皮肉、鲜血，已然化作齑粉。
<br/>　　惊蛰剑身后，有一参天大树。
<br/>　　同样，大树表面无损。
<br/>　　但萧逸手指轻弹树身，哗啦一声…
<br/>　　整棵大树，顷刻化作冰粒齑粉。
<br/>　　“冰暴噬魂，果然厉害。”萧逸暗暗吃惊。
<br/>　　冰尊圣纹诀、魂念诀、冰暴噬魂，这，是萧逸当年从冰尊者的灵识中得到的三门手段。
<br/>　　而其中，冰暴噬魂，作为冰尊者手中最强的魂技，却一直是萧逸修炼不得为之遗憾的强大手段。
<br/>　　原因，便在于冰鸾剑根本无法修炼出冰属性魂力。
<br/>　　不，准确来说，是修炼出来了，但又自动吸走了。
<br/>　　故萧逸根本没有冰属性元力去施展这冰暴噬魂。
<br/>　　直至之前从金魂殿主手中得到那大量的‘魂丹’，萧逸才有了施展的办法。
<br/>　　魂丹，简单来说，便是魂师的丹药。
<br/>　　这就像武者的寻常元力丹药。
<br/>　　寻常元力丹药，武者战斗中吞服，能快速补充元力。
<br/>　　而魂丹，则是魂师用作补充魂力的丹药。
<br/>　　不过魂丹太过稀有，其创造者，是魂殿上古时期的一位总殿主。
<br/>　　而魂丹的丹方，一直保存在魂殿总殿。
<br/>　　即便是药尊殿那边也没有。
<br/>　　唯魂殿执事以上头衔，方能每月在魂殿主殿、分殿，领取一定数量的魂丹。
<br/>　　普通魂师，则需要任务点兑换。
<br/>　　以往，萧逸也想过魂丹的事。
<br/>　　若他想要，大可去和魂殿的魂师或者殿主之流兑换。
<br/>　　只要开价足够高，即便魂丹对魂师来说极为珍贵，也会有魂师愿意换的。
<br/>　　不过，以往的萧逸，自己本就缺修炼之物，便没去换。
<br/>　　他毫不怀疑，冰暴噬魂这门高深魂技，所需要的魂力，必然极其庞大。
<br/>　　故，魂丹太少，根本没用。
<br/>　　以往，他便打消了念头。
<br/>　　而今，他入了魂殿，便是两说了。
<br/>　　他入魂殿，原因之一，也是为了这魂丹。
<br/>　　萧逸取出一粒魂丹，吞服而下，细细感知了一番。
<br/>　　魂丹入腹，快速进入小世界，化作魂力。
<br/>　　但同时，冰鸾剑也在快速吸收着这股魂力。
<br/>　　萧逸暗暗无奈，散去了感知。
<br/>　　魂力，会作为冰鸾剑修复裂痕的力量，被自动吸收。
<br/>　　也就是说，萧逸无法率先吞服这些丹药了，否则，会被冰鸾剑悉数吸去。
<br/>　　这些魂丹，只能在战斗之时使用，一瞬间大量服用，再一瞬间将所有魂力爆发出来。
<br/>　　萧逸看了眼手中乾坤戒，里头原本整整十瓶魂丹，如今，只剩五瓶。
<br/>　　施展冰暴噬魂的消耗，着实庞大。
<br/>　　不过其威力，也相当可怕。
<br/>　　萧逸看着周遭如今覆盖百里的冰暴罡风，再次吸了口凉气。
<br/>　　魂技，需要以武魂施展。
<br/>　　冰暴噬魂，乃是冰尊者手中的最强魂技。
<br/>　　加上萧逸体内的最强武魂冰鸾剑施展而出，若他拥有足够的魂力的话，其威力…
<br/>　　萧逸甚至毫不怀疑，若他当初拥有足够的魂丹的话，在血月大阵时，他怕是把镜花水月四个老家伙杀个十遍八遍都不难。
<br/>　　“要是能把魂丹的丹方捞出来…啧啧。”萧逸想了想，笑笑。
<br/>　　当然，他也只敢想想。
<br/>　　魂丹丹方，在魂殿总殿之内。
<br/>　　胆敢染指，便等着天下魂师和魂殿的无休止追杀吧。
<br/>　　萧逸摇了摇头，取了惊蛰剑手中乾坤戒，随后御空飞离。
<br/>　　这肆虐百里的冰暴罡风，估计还要吹袭个数天才会消散，萧逸也懒得理会了。
<br/>　　……
<br/>　　数天后。
<br/>　　高空中，萧逸化作一道黑色流光，疾速飞行。
<br/>　　“应当差不多要出中部范围了。”萧逸自语一声，暗暗思量。
<br/>　　“嗯？”
<br/>　　恰在此时，萧逸眉头一皱，目光直视下方。
<br/>　　下方，是一片深山野林。
<br/>　　嗖…
<br/>　　萧逸的身影，如若一只无声无息的鬼魅，就此落下。
<br/>　　身影，藏于暗处。
<br/>　　前方，数十个一身黑袍的怪异武者，聚拢着。
<br/>　　而在这群黑袍武者之前，一个同样一身黑袍，但面容阴蛰的武者，似在不悦地呵斥着些什么。
<br/>　　那人，是个熟人。
<br/>　　正是毒袍。
<br/>　　世事，果真巧，萧逸也没想到，自己不过赶个路，打算去叶流那小子那边喝个喜酒，却生生成了一段千辛万苦之行，一路上又屡遇到熟人。
<br/>　　前方。
<br/>　　“什么？”毒袍的语气，阴冷中，带着极度的不满。
<br/>　　“又被他们逃了？”
<br/>　　“这已经是天藏学宫弟子第二次逃脱了，…”
<br/>　　毒袍的语气，愈发阴沉。
<br/>　　阴蛰的眼眸中，甚至已泛过一道幽光。
<br/>　　一众邪修，面露惊慌，连连单膝跪下，“护法息怒。”
<br/>　　“那顾非凡实力极强，现今恐怕一般的长老都非是他的对手。”
<br/>　　“哼，一群废物。”毒袍冷哼一声。
<br/>　　一个个邪修，头颅低下，大气不敢喘。
<br/>　　咔咔咔…
<br/>　　毒袍的拳头，握得劈啪作响，“这是水姑娘亲自交待的事，出不得差池。”
<br/>　　“只能本护法出手了…”
<br/>　　“嗯？谁？”毒袍忽然眼眸一冷，看向不远处一棵大树之后。
<br/>　　萧逸的身影，缓缓走出。
<br/>　　边走，宽大黑袍之下，萧逸也在暗暗疑惑，又是水凝寒？
<br/>　　且，前些天，天藏学宫队伍遇到邪修，显然不是偶然，而是邪君府要对他们下手。
<br/>　　“阁下是？”毒袍眼眸一眯。
<br/>　　毒袍心头，一阵惊骇。
<br/>　　这浑身黑袍的怪人，到底在暗中藏了多久？
<br/>　　以他的实力，竟迟迟没有发现？
<br/>　　事实上，毒袍不知道的是，若萧逸不愿，毒袍等一众邪修，根本发现不了他。
<br/>　　在幽魂面具的隐匿气息下，加上萧逸自己的一身潜行本事，别说毒袍，就是等闲的传奇强者，也发现不了他。
<br/>　　不过是萧逸自己发出声响，自己走出来罢了。
