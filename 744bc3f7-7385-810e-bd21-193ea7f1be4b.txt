<br/>　　只见他缓缓抬起手臂，虚手一抓，

<br/>　　那道骇人的赤色刀芒，竟然瞬间破碎。

<br/>　　圣器被毁，屠苏玄也遭到了反噬，不由得喷出一口老血。

<br/>　　“彭！”
<br/>　　屠苏玄周身的那道外道神像，几乎连一秒钟都没有撑到，便瞬间爆碎。
<br/>　　“啊！！”
<br/>　　伴随着一道不甘的惨叫声，
<br/>　　屠苏玄直接被叶尘这一掌，狠狠拍落了下去。
<br/>　　轰隆隆！
<br/>　　整个拍卖场，足足有十几个足球场那么大，在叶尘这一掌的威压之下，竟然被瞬间夷为平地。
<br/>　　地面上，
<br/>　　出现了一个近百米深的巨坑！
<br/>　　屠苏玄整个人，完全陷入巨坑底部，生死不明。
<br/>　　嘶~~
<br/>　　看到这一幕的众人，全都纷纷倒吸了一口凉气，再次望向叶尘的目光中，充满着浓浓的畏惧、疑惑、震撼……
<br/>　　堂堂屠苏家的家主，半步化神境的强者，竟然被这个年轻人，一招秒杀了？
<br/>　　这，这特么什么情况？
<br/>　　叶尘身后的冷蝉衣，不由得用玉手捂着樱桃小嘴，美眸中透着浓浓的惊骇之色，
<br/>　　尽管她已经想到，眼前这个年轻人的真实实力深不可测，可是却也绝没有想到，这家伙的实力，竟然强大到了如此地步！
<br/>　　那可是屠苏玄！
<br/>　　在云丹城足以跻身前十的强者！
<br/>　　居然就这么被他打败了？
<br/>　　“不！这不可能！这怎么可能？”
<br/>　　至于另外一旁，
<br/>　　原本都已经准备为叶尘收尸的屠苏元让，更是两眼瞪得老大，拼命的摇着头，说什么也无法接受眼前发生的一切。
<br/>　　自己一向敬若神明的父亲，居然败给了眼前这个，比他还要年轻许多的家伙？
<br/>　　轰！
<br/>　　就在众人无比震撼之时，那深坑之中，忽然一阵晃动，继而一道狼狈不堪的人影，从深坑中窜出。
<br/>　　虽然叶尘如今实力又有所精进，可是在没有施展逆生百转的情况下，还不足以如此轻易，就能把半步化神境的屠苏玄打死。

<br/>　　随着叶尘一声暴喝，才刚刚奔出百余米的屠苏玄，陡然感觉到一股强悍到无法抵挡的力量，硬生生将他又拉扯了回去。

<br/>　　一道凌厉无比的紫色剑芒在空中划过，瞬间便洞穿了屠苏玄的脑袋。
<br/>　　屠苏玄整个人连同神魂，尽数被这一道剑芒所灭！
<br/>　　屠苏玄，死！！
