　　在白家家主的快速安排下，萧逸、叶铭、方鹤、慕容胤分别被安排到静谧的房间中。
<br/>　　炼药师研究丹方时，非常耗神，受不得打扰。
<br/>　　萧逸安静地坐在一个玄色蒲团之上，手上拿着一张丹方。
<br/>　　这是白家家主特地准备的蒲团，有凝神静心之效。
<br/>　　那丹方，自然就是四品巅峰丹药，雪魄丹的丹方。
<br/>　　萧逸看了几眼，只觉脑袋里一片浆糊，压根儿没看明白。
<br/>　　“这个世界的炼药之术，还真是晦涩难懂，玄奥至极。”萧逸无奈地摇摇头。
<br/>　　不是他愚钝，只是四品丹方，已经超出了他现在所懂的炼药知识。
<br/>　　他现在不过是个二品炼药师，自然看不懂四品丹方。
<br/>　　若是换了二品丹方，他随便看两眼就能研究透彻。
<br/>　　当然，他早就料到了如今的情况，也没打算靠自己去参悟雪魄丹丹方。
<br/>　　蓦然一笑，他的双眼忽然变为了一黑一红。
<br/>　　太阴太阳之眼陡然而现，只要配合冰鸾剑，剑身上的神秘符字，便会自动领悟丹方。
<br/>　　不过，这一次他没有将冰鸾剑凝聚出来。
<br/>　　这里毕竟是白家，强者众多，要是暗中有人窥探，那便麻烦了。
<br/>　　幸好冰鸾剑已是他的武魂，他内视体内一番，同样可以看清剑身。
<br/>　　只不过，这更为耗神，也需更认真地去观看。
<br/>　　将雪魄丹丹方上的文字记于脑海中，而后观察体内冰鸾剑剑身。
<br/>　　顷刻间，脑海中原本玄奥的丹方内容，变成了再简单不过的知识。
<br/>　　半晌，当他双眼恢复正常后，已经彻底掌握了炼制雪魄丹的步骤和注意事项。
<br/>　　只需拿了材料，他便能炼制出雪魄丹。
<br/>　　“这才过了一小会儿时间，倒也不急。”萧逸暗暗想着。
<br/>　　这时，他从乾坤袋中拿出了先天丹的丹方。
<br/>　　之前大长老给他的炼药心得中，有数个三品丹方，以及三品炼药师的大量知识。
<br/>　　他一直没怎么看过，一来是没时间，这三个月不是赶路就是清剿山匪。
<br/>　　二来是感觉无用，先天丹有杂质，不能多服，也懒得浪费时间去炼制了。
<br/>　　趁着这会儿有时间，闲着也是闲着，便拿出来研究一下。
<br/>　　同样是双眼一变，内视体内冰鸾剑。
<br/>　　先天丹的丹方变为了简单的内容，轻松便掌握了。
<br/>　　如今，只要他愿意，随时都可以炼制先天丹。
<br/>　　同一时间，脑海中蓦然出现了一个新的丹方，这是冰鸾剑创造的丹方。
<br/>　　“五行丹。”萧逸暗暗说着，并散去了太阴太阳之眼，双目恢复正常。
<br/>　　据脑海中丹方的介绍，这五行丹是提升武者整体素质的丹药。效果如何，暂时不得而知。
<br/>　　只不过，既然是冰鸾剑衍生出来的丹方，必然非凡物。
<br/>　　九转炼体丹、金脉丹，之前已经给过自己足够的惊喜。
<br/>　　炼制五行丹的材料，全是珍贵的天材地宝，最弱都是三品药材，四品药材也不在少数。
<br/>　　幸好，自己之前在狂血玄君洞府内，得到了许多天材地宝。
<br/>　　这数月清剿山贼，也缴获了不少宝贝。
<br/>　　几乎所有材料自己都有，唯独只缺一种，名为六宝玲珑参的东西。
<br/>　　“六宝玲珑参？”萧逸暗暗皱眉，这东西他大概听说过，是一种极珍贵的天材地宝。
<br/>　　“怕是一般商行是没得卖的，以后去一趟猎妖殿看看吧。”萧逸暗暗说着，而后没有再多想。
<br/>　　如今材料缺一种，多想也是无用。
<br/>　　不过，他倒是想起来。自己刚才看雪魄丹丹方时，冰鸾剑并没有衍生出新的丹方。
<br/>　　或许，只有在参悟淬体丹、后天丹、先天丹，这些能提升武者修为的丹药时，才会衍生新的丹方吧。
<br/>　　若是以后看洞玄丹的丹方，估计也会衍生出新的丹方。
<br/>　　淬体丹、后天丹、先天丹、洞玄丹、破玄丹.....都是各个品级中的基础丹药，能小幅度提升武者修为，但含有杂质。
<br/>　　将先天丹放回乾坤袋，而后拿着雪魄丹丹方随意看了几眼，萧逸便开始小憩起来。
<br/>　　大概过了一个时辰，他才淡淡地走出了房间，回到了庭院中。
<br/>　　庭院中，白家主、白管事和一众武者都在等候着。
<br/>　　“看，易霄率先出来了。”
<br/>　　在场武者低呼一声。
<br/>　　“这才过去一个时辰，真的能将雪魄丹丹方研究透彻？那可是四品巅峰丹药阿。”
<br/>　　“若这是真的，这易霄的炼药本事可就相当厉害了。”
<br/>　　在场武者们议论纷纷。
<br/>　　一旁的白管事也沉声问道，“易霄，你可是真的已经研究透彻了，可有把握？”
<br/>　　萧逸翻了个白眼，没有理他。
<br/>　　半个时辰后，叶铭也走了回来。
<br/>　　“哦？”叶铭来到易霄身边，惊讶道，“易霄小兄弟如此快便悟透丹方了吗？”
<br/>　　再过了半个时辰，方鹤走了回来。
<br/>　　他前脚刚来，没过几分钟，慕容胤后脚便也来了。
<br/>　　慕容胤见自己是最迟之人，顿时脸色一冷，暗道，“该死，这三人竟然比我悟透得更快，这岂不是说他们的炼药本事比我强？”
<br/>　　“哼。”慕容胤暗哼了一声，“方鹤和叶铭也就罢了，那易霄，估计只是打肿脸充胖子。”
<br/>　　这时，白家家主见四人已经回来，才问道，“诸位可有把握？”
<br/>　　慕容胤抢先回答道，“白家主请宽心，雪魄丹丹方，我了然于心，不出意外，足有五成以上的把握。”
<br/>　　“五成？”白家家主一喜，对于从未炼制过雪魄丹的四品炼药师来说，五成已经是很高的几率了。
<br/>　　“慕容胤执事。”白家主抱了一拳，道，“若雪魄丹真的炼制成功，你当初拜托我之事，我便应允下来。”
<br/>　　“如此甚好。”慕容胤顿时满脸笑容，更是重重地对着白家主行了一礼。也不知是什么事，开心成这样。
<br/>　　一旁的方鹤见状，不屑地瞥了他一眼，冷声道，“我有六成把握。”
<br/>　　随后，不顾慕容胤发黑的脸色，转过头看向叶铭，问道，“叶铭小兄弟你呢。”
<br/>　　“我？”叶铭轻声道，“七成左右吧。”
<br/>　　“七成？”方鹤顿时一惊，笑道，“不愧是裂天剑派炼药堂的首席弟子，厉害。”
<br/>　　下一秒，三人看向易霄。
<br/>　　白家主也出言问道，“易霄，你呢？”
<br/>　　萧逸耸耸肩，淡淡道，“还是那句话，不比慕容胤差。”
<br/>　　“小子，你...”慕容胤原本就发黑的脸色，顿时黑得发紫。
<br/>　　白家主摆摆手道，“既然四位都有把握，那么便开始炼药吧。雪魄丹，就仰仗四位了。”
<br/>　　四人连忙道，“白家主客气。”
<br/>　　正当白家主准备让四人去安静的炼丹房时，慕容胤却站了出来。
<br/>　　“白家主，不必去炼丹房。”慕容胤自信地说道，“就在这里炼药即可。”
<br/>　　“就在这里？”白家主眉头一皱，道，“不怕被打扰吗？”
<br/>　　慕容胤摇摇头，道，“雪魄丹的炼制过程和步骤，我早已了然于心，自然不怕被打扰。”
<br/>　　“白家主，那方某也就地炼药吧。”方鹤立刻说道。
<br/>　　“嗯。”白家主点点头，既然炼药师说没问题，他一个外行也不好说什么。
<br/>　　“叶铭小兄弟你呢？”白家主问道。
<br/>　　叶铭仍旧是轻声道，“难得慕容胤执事和方鹤大师有雅兴当场炼药，在下自然不会扫兴，也一并就地炼药吧。”
<br/>　　“易霄小兄弟呢？”白家主看向萧逸。
<br/>　　萧逸翻了个白眼，暗道，他们三个都就地炼药了，我还有得选吗。
<br/>　　心中腹诽，但嘴上还是说道，“我也就地炼药吧。”
<br/>　　慕容胤心中冷笑一声，暗道，“哼，早就知道你们三人也会如此。待会儿我当众炼制出雪魄丹，证明炼药本事比你们强时，我看你们还有什么脸面对我冷嘲热讽。”
<br/>　　说白了，他选择就地炼药，只是想出风头，享受在场武者崇拜的目光。
<br/>　　其余三人的目的则不然。
<br/>　　方鹤只想压住慕容胤。
<br/>　　叶铭不知出于什么原因，永远一副神态自若的样子。
<br/>　　萧逸纯粹是无奈，当然，也无所谓。
<br/>　　庭院中，一众武者散开，留出中心地带偌大的地方给四人。
<br/>　　四人彼此隔着数米距离，待得白家主拿出八份材料，四人拿出炼药炉，正是开始炼药。
<br/>　　慕容胤的动作驾轻就熟，快速地投放着材料。
<br/>　　同时，一道火红色的藤蔓凝聚而出，带着一阵噼里啪啦的声音，以及熊熊火焰。
<br/>　　这是霹雳火藤，是一种遇火便会快速大规模燃烧的神奇藤蔓，用作炼药助燃，效果极好。
<br/>　　霹雳火藤武魂，品阶为绿色。
<br/>　　一旁的方鹤也显得游刃有余，在他身前，有一只虚无的灵鹤，正不断地对着炼药炉喷火，火势把握得几近完美。
<br/>　　这是火鹤武魂，品阶绿色，操控的灵鹤之火带有灵性，令得他能游刃有余地掌控整个炼药过程。
<br/>　　叶铭倒显得平平无奇，也不知用的是什么武魂，但显然一副胸有成足的样子。
<br/>　　反倒是萧逸，一阵紫炎打出，威势骇人。
<br/>　　“不愧是紫炎易霄，他的火焰很强，也不知是什么武魂有如此力量。”
<br/>　　在场武者认不出，只得暗暗惊讶。
<br/>　　事实上，萧逸不得不使用紫炎。
<br/>　　雪魄丹的材料，都不是低品药材。以他如今的修为，若是用普通火焰，也不知要烧到什么时候才能将材料中的药力完全提炼出来。
<br/>　　只能借助紫炎了。
<br/>(本章完)
