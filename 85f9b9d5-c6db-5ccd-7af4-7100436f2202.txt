　　第375章
<br/>　　夷族有两样东西最拿得出手，一是各种奇技淫巧的小物件，别开生面的思维；二就是女人，来自夷族，裹夹着异域风情的女子，一直是帝国上层人物的追捧之物。
<br/>　　据说，明帝宫里就有来自夷族的胡妃。
<br/>　　这两样物事，对于后者程大雷并不在意……呃，应该不在意吧。但对于前者，恰恰是他所需要的。
<br/>　　程大雷看着面前这个英俊得令人发指的男人，凯恩·道尔同学对程大雷隐瞒的怕不止一件喔。
<br/>　　“呔！”程大雷突然发怒：“我天朝上邦，包容万物，却绝不容忍欺骗，既然你不肯实话实说，看来我只要把你卖了做奴隶。”
<br/>　　凯恩吓得脸色苍白，当然，他皮肤本就白。听程大雷如此一说，口中忙道：“程当家，我对你什么都说了，不敢有一句欺瞒。”
<br/>　　“呵，哈哈哈，你当真以为我是好欺骗的么。”
<br/>　　程大雷一阵冷笑，笑得凯恩不寒而栗。徐神机在一旁道：“你有什么事，就实话实说，怎么说都可以，我们大当家最讨厌别人骗他。”
<br/>　　凯恩叹了口气，这才把实话说出口，大体和刚才说得差不多，只是有些细节他隐瞒了。
<br/>　　诺顿王朝对异教徒的容忍度为零，这一点就没办法和帝国相提并论了。在帝国只要你不造反，其他事都好商量。
<br/>　　当然，帝国也有各种各样的愚夫愚妇，信仰各种各样的神明。可都是比较务实的，那个神仙管用拜那个，暗地里编排其神仙来，也是一点不留情。
<br/>　　这一点，在诺顿王朝就做不到了，谁要敢在诺顿王朝说一句神明的不是，就要被放在绞刑架上处死的。
<br/>　　诺顿王朝信仰的神灵是万能的造物神，在教廷的理念中，造物神管理一切，上到星辰，下到一草一木。可是，在底层还是有人信仰各种各样的神灵。
<br/>　　比如：大地之神、生命之神、寒冰之神、火焰之神。
<br/>　　信仰造物神的可以称为旧教，信仰诸神的可以称为新教。新教在王国已经是一股不小的势力，并不是散兵游勇，甚至暗中还有王朝的支持。
<br/>　　毕竟，王权也不愿意屈服在神权下。其间事情很是复杂，凯恩也没有对程大雷详细说，最后的结果，反正是新教失败了。
<br/>　　新教的残余势力，在年轻的首领带领下，突破层层难关，克服艰难的自然环境，逃到了东方。
<br/>　　只因为他们听说东方有个庞大的帝国：富裕、繁华、和平、拥有强大的文化。更重要的一点是：没有宗教。
<br/>　　所以这伙人来帝国的目的是：传教。
<br/>　　程大雷有些无语，这些异域人对传教是有怎样的执着啊。
<br/>　　如今新教的人已经很少，大概就是五六十人，包括各种职业：铁匠、皮匠、炼金术师、药剂师、航海家、冒险家、酿酒师……甚至还有魔法师。
<br/>　　程大雷听着听着眼睛就亮了起来，这些人会不会携带帝国没有的技术。
<br/>　　“这个……诺顿王国还有魔法师？”
<br/>　　“是。”
<br/>　　“是真的么？”程大雷好奇的问道。
<br/>　　“当然是真的。”诺顿激动起来：“他们以灵魂祭祀诸神，拥有诸神赐予的能量，可以召唤雷电，起死回生。”
<br/>　　“你亲眼见过？”
<br/>　　“我亲眼见过，我以我的信仰起誓，我是不会撒谎的。”
<br/>　　“那你也是信仰风之精灵的，能不能给我展示一下？”程大雷好奇问。
<br/>　　“这个……我的职业是吟游诗人，不懂魔法的事。”
<br/>　　程大雷撇撇嘴，各个民族都有各种各样的神职，比如帝国也有道士、和尚、巫婆。传说中道士能起死回生，和尚能驱魔，巫婆可以请神上身……如果随便问一个人，他们肯定比凯恩说得还真，至于是不是真的，那就看每个人的智商程度了。
<br/>　　“好啦，你们的人可以住在琴川关。”程大雷道：“你们的人现在在哪儿？”
<br/>　　“他们现在在一个山谷中，因为附近都是戎族……”
<br/>　　程大雷点点头，也理解凯恩的话，戎族碰到这帮人，还不直接男的抓了当奴隶，女的做老婆。草原上的这些野蛮人，可没有心情去了解你信仰什么神灵。
<br/>　　其实不仅仅是戎族，帝国对夷族也绝对称不上友善。当然，帝国还是很包容的，可耐不住夷族的女人漂亮啊。谁不希望家里有个胡姬，身娇体媚，带出去也有面子。
<br/>　　听凯恩说了山谷的具体位置，距离琴川关倒也不远。据凯恩说，他们的情况不是很好，穿越无边沙漠，死的死伤的伤，现在又缺水又缺粮。
<br/>　　事情倒是不可耽搁，程大雷让人把秦蛮喊来，带些干粮清水，跟着凯恩去接人。顺便让徐神机安排出几座院子，等这些人来了居住。
<br/>　　“这个……程当家要不要多带一些人去？”凯恩忐忑道。
<br/>　　“干嘛？”程大雷不解问，派了几十个人去，还有几辆马车，把腿脚不便的接过来，这些人已经足够了。
<br/>　　“万一我们路上碰到戎族……戎族可是好凶的？”凯恩的话里不无担忧。
<br/>　　程大雷瞥了他一眼，表情有些无语。
<br/>　　“好啦，你放心，只要是我们琴川关的人，在草原上是无人敢劫的。”徐神机道：“现在整个琴川，有谁不知道我们蛤蟆大王的名字。”
<br/>　　程大雷摆摆手：“低调，低调，不要搞得这么浮夸么。”
<br/>　　对于将要到来的人，程大雷心里还是很期待的。不知这群来自诺顿王国的人，会给蛤蟆寨带来什么新鲜血液。
<br/>　　万一人家真有魔法师呢……传说不可尽信，也不可不信，如果魔法是真的，那么这个世界的力量层次，就要重新估计了。
<br/>　　程大雷隐隐有些期待，但估计……够呛。
<br/>　　天已晚，程大雷拐进了樊梨花的院子，如今她就住在将军府，距离程大雷也颇近。
<br/>　　叩门而入，看见樊梨花时，程大雷隐隐就有些头疼。桌上摆着饭菜，她一动不动，正抱着琵琶，由柳芷教她学弹。
<br/>　　见到程大雷进来，柳芷微微施礼，识趣的退了出去。
<br/>　　看着樊梨花手中的琵琶，程大雷真不知说什么好。
