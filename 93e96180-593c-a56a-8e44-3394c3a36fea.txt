　　萧逸边看着脑海中的文字，边快速参悟起来。
<br/>　　半晌，他大概知晓了这门‘血刹魔经’是怎么回事。
<br/>　　这是一门气血一道的功法。
<br/>　　他自己本就有修习气血一道，不过倒没有这方面的功法，只有武技。
<br/>　　若修习了这门功法，他的气血一道，必然威力更强。
<br/>　　另外一点，抽调这颗血色圆球的速度，也必然会快许多。
<br/>　　虽然听这门功法的名字，便知不是什么好东西。
<br/>　　但萧逸对此，早已不在乎。
<br/>　　实力二字，是他现今最为看重的东西。
<br/>　　无论是数月后的剑域盛事，还是一年后的古帝之墓开启，抑或是三年之约，他都需要足够的实力。
<br/>　　倒是冷焱剑内的剑灵，看来此番血光洞府一行之后，还需要重新审视一番才是。
<br/>　　因为，在他刚才的大概参悟中，这门血刹魔经的层次，远比他想象中的高。
<br/>　　除却‘修罗战体’外，他所修习的功法，竟是几乎没有一门能比得上这‘血刹魔经’。
<br/>　　萧逸脸上闪过一丝凝重。
<br/>　　一旁，公孙火舞见萧逸脸色不断变幻，蓦地脸色怪异了起来，“小贼，你是不是魔怔了，你别吓我。”
<br/>　　“我听我爹说，气血一道，极其危险，动辄心魔爆发，身死道消。”
<br/>　　“你是不是抽调这些气血力量产生心魔了？”
<br/>　　“要不你休息一下？”
<br/>　　公孙火舞无法听到萧逸与剑灵间的传音，脸色蓦地担忧了起来。
<br/>　　“我没事。”萧逸摇了摇头。
<br/>　　回答罢，萧逸彻底参悟了起来。
<br/>　　一边抽调气血力量，替顾怜星修补血脉，一边参悟。
<br/>　　对萧逸而言，一心二用，只是小事。
<br/>　　以他的天赋，这算不得什么。
<br/>　　一天后。
<br/>　　萧逸彻底从参悟中醒来。
<br/>　　一手抽调气血力量，另一只手，则一把握住了冷焱剑。
<br/>　　轰…
<br/>　　几乎是一瞬间，主府之内，气血力量轰然爆发。
<br/>　　那颗血色圆球内的力量，以一股恐怖的速度，被极快地抽调着。
<br/>　　“好强，好快的速度。”公孙火舞瞪大了眼睛。
<br/>　　冷焱剑上，霎时一阵异动。
<br/>　　“血刹吸元之法？厉害，你这小子是怪物吗？”
<br/>　　“一心二用之下，仅用一天便彻底参悟血刹魔经？”
<br/>　　剑灵的话，没有再说下去。
<br/>　　因为，此时，一股股精纯浑厚的气血力量，已源源不断地进入冷焱剑剑身之内。
<br/>　　萧逸双手齐出。
<br/>　　两股气血力量，分别进入剑身以及顾怜星体内。
<br/>　　毫无疑问，现今的抽调速度，比之前快了数十倍都不止。
<br/>　　如果说，之前抽调吸收的气血力量是一丝丝。
<br/>　　那么，现今抽调而出的气血力量，便是一股股，浑厚至极。
<br/>　　萧逸全力抽调着血色圆球内的气血力量。
<br/>　　不过，即便是他这般惊人速度下，血色圆球仍旧不见有丝毫影响，一如之前那般血色刺眼。
<br/>　　不难想象，这血色圆球内的气血力量，绝对澎湃到一个惊人的地步。
<br/>　　被抽调而出的两股气血力量，一股进入顾怜星体内，自不必多说。
<br/>　　而另一股，虽然进入冷焱剑剑身，但还是会像以往那般，有一小部分会通过剑身，进入他体内。
<br/>　　萧逸明显可以感知到，他体内的那颗‘血丹’，正不断快速‘成长’。
<br/>　　萧逸皱了皱眉，但半晌，选择不理会。
<br/>　　他这般全力抽调着。
<br/>　　却没注意到，广场处，八位天骄抽调血色力量的速度已然停下。
<br/>　　这证明着，他们八人已经吸收够足够的气血力量，本身拥有的血脉之力，也已激发到极致。
<br/>　　广场边缘，一位位长老，闪身而入。
<br/>　　既然这八人的血脉之力已然发挥到极致，那么，现在便该是正式开始武魂第二次觉醒之时。
<br/>　　嗖…嗖…嗖…
<br/>　　三十位长老，各自闪身来到自家天骄身旁。
<br/>　　“引。”一声声低喝，同时爆发。
<br/>　　这八人，血脉之力已发挥到极致，武道圣丹也已消化。
<br/>　　体内力量，正是巅峰之时。
<br/>　　这时，这一位位圣皇境巅峰长老，为他们引导力量，冲击觉醒即可。
<br/>　　武者觉醒第二次武魂，之所以需要武道圣丹。
<br/>　　便是因为需要足够的天地武道力量。
<br/>　　那八颗武道圣丹，层次绝对超过了武道大能。
<br/>　　虽不是这些圣皇境巅峰长老的武道圣丹，但这并不影响什么。
<br/>　　他们的引导，只是凭借他们的圣皇境巅峰修为，以及这个境界的武道手段，让得这八人体内的力量充分被发挥罢了。
<br/>　　毕竟他们的力量掌控，绝对远强于这八位天骄。
<br/>　　只要这八人体内已准备好充足的力量，以这一位位长老的境界，引导力量去冲击觉醒，并非难事。
<br/>　　大半天后。
<br/>　　嘭…嘭…嘭…
<br/>　　一连八道剧烈的气势爆响声，响彻洞府。
<br/>　　一连八道狂猛的气势，席卷周遭。
<br/>　　“成了。”一位位长老，面露喜色。
<br/>　　而原本闭目的一位位天骄，亦霎时睁开了眼睛，眼中精光暴现。
<br/>　　那一双双桀骜的眸子中，透漏着的，是无比的自信。
<br/>　　那一股股萦绕在八人身体周遭的武道力量，证明着，这八人，已然踏入了圣皇境，已是武道皇者。
<br/>　　如此年轻的武道皇者，足以让人惊骇，让人羡艳不已。
<br/>　　“恭喜少府主，武魂觉醒成功，踏入武道皇者。”
<br/>　　一位位长老，连连拱手恭贺。
<br/>　　另一边。
<br/>　　主府之内，萧逸仍旧快速吸收着血色力量。
<br/>　　在他的感知中，顾怜星的血脉，已被修复九成以上。
<br/>　　再过一阵，便能彻底修复。
<br/>　　“嗯？”忽然，萧逸皱了皱眉头。
<br/>　　他靠近血色圆球的指尖，忽然一阵异动。
<br/>　　“什么东西，好可怕的吸力。”萧逸脸色一惊。
<br/>　　他以指尖抽调血色力量。
<br/>　　此时，抽调速度戛然缓慢，血色圆球之内，似有什么东西，反向地吸摄着。
<br/>　　“给我出来。”萧逸心头低喝一声。
<br/>　　修炼血刹魔经后的萧逸，对气血一道的掌控无疑达到惊人的地步。
<br/>　　在他的强行抽调下，一颗浑圆的血色珠子，缓缓从血色圆球中心被抽调而出。
<br/>　　“嗯？”萧逸皱着眉。
<br/>　　身后，主府之外。
<br/>　　八位天骄，事实上早已发现了主府内的萧逸三人。
<br/>　　八人本还沉浸于成功觉醒的喜悦中，不以为意。
<br/>　　但，当看到主府之内，那被缓缓抽调而出的血色珠子，却陡然脸色大变。
<br/>　　“怎么可能…”
<br/>　　“那是血舍利。”
<br/>　　八人脸色的喜悦，霎时化作了惊色。
<br/>　　“上古至宝，血舍利？”一位位长老，同样瞪大了眼睛。
<br/>　　与此同时，一抹冷色，涌上一位位长老的脸庞。
