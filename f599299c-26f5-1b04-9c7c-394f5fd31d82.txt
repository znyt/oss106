　　第534章
<br/>　　黄老汉脸色煞白如纸，冷汗啪啪直落。他踉跄跪倒在地：“两位好汉，一定要救老汉一救。”
<br/>　　程大雷轻轻摇摇头：“你想火中取栗，就要准备好引火烧身，尸骨无存。”
<br/>　　轰！
<br/>　　此刻黄家正门和后门一声巨响，官兵和土匪同时闯破前后门杀了进来。黄家老小，但凡遇见的，无非就是个死而已。黄为善看着这一幕，心如刀绞，口中不住道：“各位兵爷，他们是我的家人，不是土匪呐。”
<br/>　　方石隆连搭理他的意识都没有，一路从正门闯到大厅，恰好朱刚等人也从后门杀进来。双方势力一前一后，将大厅团团围住。
<br/>　　“哪个是程大雷！”
<br/>　　方石隆拎着大刀，刀刃上的血往下滴。他走到大厅前，见四面亮着火烛，大厅中央摆着一张八仙桌。
<br/>　　桌前坐着两条大汉，正旁若无人的喝酒吃菜。而在大厅一角窝着几个人，此刻脸色苍白，挤在一起瑟瑟发抖。
<br/>　　张猛带人从后门杀到，肩头扛着一条大枪，口中哇哇大叫：
<br/>　　“程大雷在哪里，程大雷在哪里，快快还我哥哥命来。”
<br/>　　黄老汉缩在柱子下，此刻已没有力气站起来。此刻大厅被官兵和土匪围了，空气中有淡淡的血腥气，这些都是黄家人，或死在官兵手中，或死在土匪手中。比如那个叫雀儿的丫鬟，现在已经尸首分离，倒在血泊之中。
<br/>　　八仙桌前的两条大汉，仍旧喝酒吃肉，似乎没看见包围的敌人。
<br/>　　方石隆眉头皱起，拿出一份通缉程大雷的画影图形，比对着厅上两条大汉。看看这个也像，看看另外一个也差不太多。
<br/>　　“喂，你们那个是程大雷。”
<br/>　　张猛扛着大枪，忽然哇呀大叫一声，一枪向八仙桌砸来。
<br/>　　间不容发之极，一个铜酒壶拦在大枪之下。壶口被砸得变形，但的确挡住了这宛若有千钧之力的一枪。
<br/>　　刘发财看着程大雷，张猛也看着程大雷，几乎所有人视线都落在程大雷身上。
<br/>　　程大雷甩开肩膀，同时也将张猛的大枪甩开，他将面前酒杯倒满，把歪斜的酒壶搁在桌上。
<br/>　　“某就是你们口口声声要杀的人，某家在这里，你们谁来杀我？”
<br/>　　刘发财的眼睛一瞬间瞪大，黄老汉的眼睛也瞪直了，口中呐呐自语。
<br/>　　“我便知道，我早该知道！”
<br/>　　“狗贼，还我哥哥命来！”
<br/>　　张猛哇呀叫了一声，倒提黑铁枪，向着程大雷胸口扎来。
<br/>　　程大雷也不见如何动作，便握住了枪杆。张猛的枪如同陷入大山中，拔，拔不出来，扎，扎不进去。
<br/>　　“好娃娃，你脾气太急了。”
<br/>　　程大雷喝了一声，手腕一震，张猛就感觉一股巨力传来，胳膊一阵酸麻，双手竟不自觉松开，铁枪到了程大雷手中。
<br/>　　鹰嘴岭的朱刚几人都倒吸了一口凉气。张猛天生好皮囊，皮肤像雪瓷一样白，两腮肥嘟嘟的像庙里的菩萨身边的娃娃，多少大姑娘小媳妇看见他挪不开眼。不过，这只是他的外表，内里的他性烈如火，残忍好杀，尤其是一身本事，山寨其他几位当家联手也对付不了他。
<br/>　　但就是这样一个白脸莽夫，却在程大雷面前一招都没走过去。
<br/>　　此人自称程大雷，现在已经无人怀疑。
<br/>　　程大雷将铁枪扎在青石板中，口中道：“想拿我的人头邀功，就要看看你们各人本事怎样了。”
<br/>　　方石隆皱紧眉头，点了三人的名字：“赵恒，周虎，方正，你们上。”
<br/>　　这三人都是他的得力手下，身上都有硬功夫。此刻握着腰刀冲进大厅。三柄刀同时向程大雷身上招呼过来。一人封程大雷的头顶，一人封程大雷的后路，一人砍程大雷的双腿，瞬间让程大雷陷入腹背受敌的境地。
<br/>　　刘发财握紧了刀柄，用刀他是行家，到现在也就剩下拼死一战这条路。
<br/>　　可还没等他出手，程大雷就抓起三根筷子，雷鸣电闪之间，筷子如毒蛇一般扎入赵恒、周虎、方正的手腕上。
<br/>　　三人惨叫连连，血从他们手腕冒出，洒在酒席上。
<br/>　　程大雷撇撇嘴，将手中的酒杯抛掉。
<br/>　　“无端扫了本当家的酒兴，呔，你们那个再上前，本当家便不会留情了。”
<br/>　　“好狗贼，你忒狂妄了些吧。”
<br/>　　张猛是少年心性，怎可服人。他拔出身上佩刀，向程大雷冲去。
<br/>　　“老九，别莽撞！”
<br/>　　长须鬼孙山起毕竟谨慎些，要拦住冒失的九弟。
<br/>　　锵啷一声，程大雷的匹夫剑出鞘，一剑便穿了张猛的喉咙，好一条白脸壮汉，身体噗通一声倒在地上。
<br/>　　“老九！”
<br/>　　鹰嘴岭几位当家气得哇呀呀爆叫，各持兵器，同时向程大雷冲了过来。
<br/>　　兖州兵这边，与赵恒三人交好的，此刻也冲进大厅。程大雷抖索起精神，一柄剑舞开，剑光所过之处，化作血光。但凡冲入程大雷剑围之人，前程也就剩个死字。
<br/>　　刘发财也拔出了刀，逼到这种地步，他也只能放手一搏。不拼命，难道伸长脖子让人杀嘛。
<br/>　　方石隆脸上的表情越发难看，大厅内，火光烛影，死的人越来越多。程大雷身在死人堆中，一剑快过一剑，一剑狠过一剑。
<br/>　　刀枪剑影，迫不到程大雷身边，全都被他一剑挡住。
<br/>　　到得后来，士兵里三层后三层围在一起，却紧绷着脸不敢上前。
<br/>　　程大雷哈哈笑了一阵，将酒壶中的酒喝光，手搭住杀红眼的刘发财。
<br/>　　“好啦，好啦，走喽。”
<br/>　　程大雷提起刘发财，将他拎着丢出大厅。他自己也从屋檐下跃出，凌空一跃抓住了刘发财的身体。
<br/>　　几个纵跃，带着他的飞过层层屋脊，来到后院的牛棚。
<br/>　　“走喽，走喽。”
<br/>　　几声呼喝，程大雷骑上黑牛，闯过了院子。
<br/>　　方石隆仰头看着这一幕，竟露出几分心向往之的意思。
<br/>　　如此气概风度，确实也不愧那帝国人人都欲杀的天下第一贼。
<br/>　　回过神，撞到朱刚等人的目光。兵匪向来不相容，今日撞在一起，非得拼出个死活不可。
<br/>　　“弟兄们，与我杀！”
<br/>(本章完)
