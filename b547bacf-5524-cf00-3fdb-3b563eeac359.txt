　　第687章
<br/>　　苏樱拨弄着手指，想着程大雷刚才无可奈何的模样，竟觉得有几分有趣。
<br/>　　以她心思，倒不是很在意程大雷扩充一下后院。程大雷毕竟没有做得太过分。只要不威胁到自己的身份，便不算什么。甚至，即便威胁到自己的身份，苏樱也未必在意。
<br/>　　她以这个世界的常理要求程大雷，但却并未按这个世界的常理来看待自己。
<br/>　　算是在程大雷面前应承下这件事，但想着自己去做这件事似乎不太合适，便打算拉着樊梨花一起去。
<br/>　　樊梨花和苏樱关系还算不错，也并没有那种争风吃醋的事情发生。原因不是因为二人都大度，而是因为二人目前和程大雷的关系都不算太好，两个苦闷人，难免就凑在一起数落程大雷的不是。
<br/>　　落玉寨目前搬到凉州城，樊梨花有了自己的事做，精气神也比之前好了许多。
<br/>　　苏樱找到她的时候，她正跨在马上，训练手下女兵骑术。虽然不指望她真训练出一支精锐骑兵，但她拿起这件事便放不下，程大雷也就索性随她去。
<br/>　　一身戎装，跨着胭脂马，手持绣绒刀，当真英姿飒爽。见苏樱过来，她纵马奔来，在将要接近苏樱时忽然勒马停下。
<br/>　　苏樱一步不动。
<br/>　　樊梨花有些吃惊：“你竟然不怕？”
<br/>　　苏樱无语的看着她，道：“你竟还如此胡闹，快快下来，我有事与你商量。”
<br/>　　其实樊梨花的潜质并不低，绣绒刀下丧敌魂，石榴裙下染征尘，估计也是传说潜力。不过程大雷舍不得她上战场，越来越养废了。在深闺大院住着，的确不是她的理想，只要在沙场之上，方才能恢复几分气力。
<br/>　　她翻身下马，道：“姐姐会骑马不会，我教你？”
<br/>　　苏樱比樊梨花年纪要小，但显得懂事一些，她摇摇头道：“会一些，会得不多。”
<br/>　　“苏姐姐想学，我也可以教你？”
<br/>　　徐灵儿骑马过来，许久不见，褪去稚气，身材长开后，也变得秀丽起来。
<br/>　　“好妹子，我回头再找你说话，今次是有事找你樊姐姐。”
<br/>　　苏樱三言两语说明来意，樊梨花对这件事没多大兴趣，相反徐灵儿却是兴致勃勃。
<br/>　　“据说那崔白玉有长安第一美女自称，妹妹不想去见见？”
<br/>　　樊梨花微微嘟嘴，心里也有些好奇。她自认为模样不如何，也是想看看谁敢号称长安第一美女，能将长安男人迷得五迷三道的又该是什么模样。
<br/>　　苏樱扯了她一把，她才答应同去，徐灵儿也想凑凑热闹。苏樱乘了一顶小轿，樊梨花与徐灵儿骑在马上，三人赶往将军府。
<br/>　　高力士一见来的是苏樱，屈膝便拜。蛤蟆城上下，对苏樱还是满尊敬的，主要是苏樱存在感不是很强，也不会同程大雷吹什么枕边风，忽悠他去做什么异想天开的事情。如此，至少就称得上贤惠了。
<br/>　　“不知二位夫人亲至，小得接驾来迟，还请夫人恕罪。”
<br/>　　徐灵儿对高力士倒不客气，直接道：“好了好了，省下这些东西，那白玉美人在哪儿，引我们去见见？”
<br/>　　“三位请。”高力士在前面引路，边走边道：“那女人不是很好对付，在长安出名的不仅有美貌，还有头脑，三位要小心。”
<br/>　　“呵，我还怕了她不成。”樊梨花冷笑。
<br/>　　苏樱走进了将军府，府内多是无家可归，被程大雷抢来成为俘虏的女人。如今，她们多已知道自己的命运，人人死气沉沉，给人一种不舒服的感觉。
<br/>　　推开两扇木门，高力士做了一个请的姿势，自己并没有走进去。
<br/>　　苏樱走进屋子，微微皱眉，她用她的能力阅读房中的信息。
<br/>　　很干净，即便成了阶下囚，对方也将屋子收拾得整整齐齐。这至少证明她们曾经过得是养尊处优的生活，而另外也证明她们有很强的求生欲。
<br/>　　一个求死之人，大概不会在意自己的居所整齐与否。
<br/>　　屋里四个女人，一个坐着，三个站在她身后。苏樱第一眼便认出了自己想见的是谁，她的美貌，便是苏樱一个女人也觉得震惊。苏樱也有些姿色，但在这个女人面前仍有些自惭形秽。
<br/>　　小家碧玉碰见大家闺秀，难免相形见绌。
<br/>　　“崔白玉？”
<br/>　　崔白玉颦起眉梢，上上下下打量苏樱一遍，道：“苏樱？”
<br/>　　苏樱忽觉有些匪夷所思：“你认得我？”
<br/>　　“闻名已久。”苏樱看向樊梨花：“这位该是樊梨花。”
<br/>　　接着看向徐灵儿，这才露出困惑眼神，摇摇头道：“这位倒是不认识了？”
<br/>　　崔白玉研究过程大雷，自然也了解他身边的女人。
<br/>　　樊梨花有种不舒服的感觉，似乎自己走入房间的第一时间，已经被对方看穿。是对方在掌握谈话节奏，而这绝不是她喜欢的感觉，可惜纵有一身武艺，樊梨花却也不知该如何是好。
<br/>　　倒是苏樱镇定些，目光掠过崔白玉看向崔白玉身后的三名丫鬟，道：“这三位应该就是犬鹿刀了。”
<br/>　　崔白玉：“夫人竟然知道？”
<br/>　　“略有耳闻。”苏樱看向三人：“这位青衫姑娘该是小鹿，这位该是小刀，这位该是小犬……”
<br/>　　小鹿三女有些吃惊，吃惊并非因为苏樱知道三人，而是她能准确的知道谁是谁。毕竟双方没照过面，只是靠外人描述便做到这一点，并不十分容易。
<br/>　　说着，苏樱笑着摇摇头：“好好一个姑娘家，为何要娶个猫啊狗啊的名字，我帮你改个名字，唤作香珠可好？”
<br/>　　终于，谈话节奏，被苏樱稍稍扳回来一些。这里是凉州城，是程大雷的凉州城，而苏樱是程大雷的夫人，她可以一句话改你的名字，也可以一句话定你生死。
<br/>　　崔白玉到此刻才起身，微微弯腰：“失敬失敬。”
<br/>　　苏樱腰背笔直，生受了这一礼。
<br/>　　“你不必怕，我这个人很好说话，没有那么大的规矩。”
<br/>　　崔白玉眉头一紧，眸子中厉色一闪而逝，双方目光碰撞，面带微笑，笑里藏刀。
<br/>　　樊梨花和苏樱面面相觑，忽地觉得，这崔白玉和苏樱竟有几分相似。
<br/>(本章完)
